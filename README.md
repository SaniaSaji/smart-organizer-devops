# Smart Organizer DevOps Project

## Overview

This project is a simple Flask application containerized using Docker and Docker Compose. It demonstrates basic DevOps concepts including containerization, multi-container applications,
and version control with Git.

## Technologies Used

* Python
* Flask
* Docker
* Docker Compose
* PostgreSQL
* Git

## Features

* Flask web application
* PostgreSQL database integration
* Docker containerization
* Multi-container setup using Docker Compose

## Learning Objectives

This project was created to learn:

* Docker images and containers
* Dockerfile creation
* Docker Compose
* Flask application deployment
* PostgreSQL integration
* Git and GitHub workflows

## How to Run

### Build and Start Containers

docker compose up --build

### Open in Browser

http://localhost:3000
